SIMPLICA - A RESPONSIVE HTML BOOTSTRAP THEME

Information:

Index.html contains the base coding of the theme. All the necessary links and scripts are mentioned in there.

Style.css contains all the custom styling of the theme. (excluding bootstrap)

Fixed.css contains all the data related to the fixed images of the theme.

Smooth.js contains all the animations. It includes code of Navigation bar fadein and fadeout, smooth scrolling and slide in review section

Others: All the necessary folders and files are included like bootstrap-4.1.3-dist etc.

img: Folder contains all the included images.
